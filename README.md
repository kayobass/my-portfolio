# 🌐 Portfolio - Kayobass

Um portfólio web moderno, responsivo e minimalista, desenvolvido para apresentar projetos, competências (stacks), carreira profissional, formação acadêmica, proficiência em idiomas e formas de contato.

---

## 🤖 Codesenvolvido com Inteligência Artificial

Este projeto foi inteiramente codesenvolvido com a assistência de **Inteligência Artificial**. A IA colaborou ativamente com o desenvolvedor em todas as etapas, desde a estruturação do código, design visual premium e customizado, até o sistema de internacionalização dinâmico.

---

## 🛠️ Tecnologias Utilizadas

Para garantir um carregamento ultrarrápido e evitar complexidades desnecessárias de builds/compilações, o portfólio foi construído usando tecnologias modernas direto via CDN:

- **[React](https://react.dev/) (v18)** — Para estruturação e componentização reativa do app (usando `React.createElement`).
- **[i18next](https://www.i18next.com/) & [react-i18next](https://react.i18next.com/)** — Para o sistema dinâmico de tradução e internacionalização (PT/EN).
- **CSS3 Moderno** — Com variáveis de design centralizadas (`:root`), efeitos de _glassmorphism_ (desfoque de fundo) e design adaptável (responsivo) para dispositivos móveis e desktops.
- **HTML5 Semântico** — Garantindo boas práticas de acessibilidade e SEO.

---

## ✨ Funcionalidades

- **Alternância de Idiomas**: Troca de idioma em tempo real (Português 🇧🇷 / Inglês 🇬🇧) sem recarregar a página.
- **Seções Dinâmicas**:
  - **Sobre**: Apresentação profissional e pessoal.
  - **Minhas Stacks**: Badges interativas com animação de hover e ícones das tecnologias.
  - **Projetos**: Cards elegantes com links diretos para o site/deploy do projeto e repositório no GitHub.
  - **Experiência Profissional & Acadêmica**: Linha do tempo limpa e organizada.
  - **Idiomas**: Nível de proficiência exibido através de texto e de uma barra de progresso com gradiente estilizado.
  - **Redes & Contato**: Botões interativos com links diretos (GitHub, LinkedIn) e atalho para envio de e-mail (`mailto:`).
- **Design Premium**: Paleta de cores escura e sofisticada (tons de preto, vermelho e bege), tipografia moderna (fonte _Inter_) e micro-animações interativas.
